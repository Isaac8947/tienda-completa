Demo de Animación de Header

Desplázate hacia abajo para ver cómo el header se oculta elegantemente y hacia arriba para verlo aparecer con una animación suave.

Características de la Animación

Scroll Arriba

El header aparece suavemente cuando haces scroll hacia arriba

Scroll Abajo

Se oculta elegantemente cuando haces scroll hacia abajo

Animación Fluida

Transiciones suaves con efectos de blur y sombra

Sigue haciendo scroll

Esta es una demostración de cómo funciona la animación del header. Mientras más contenido tengas en tu página, más útil se vuelve esta característica para mejorar la experiencia del usuario.

Ventajas:

  • Más espacio para contenido
  • UX moderna y elegante
  • Navegación intuitiva
  • Rendimiento optimizado

Características técnicas:

  • RequestAnimationFrame para rendimiento
  • Transiciones CSS optimizadas
  • Responsive para móvil y desktop
  • Respeta prefer-reduced-motion

Configuración Automática

La animación se activa automáticamente en todas las páginas que incluyan el componente. No necesitas configuración adicional.

<?php include 'includes/header-scroll-animation.php'; ?>
